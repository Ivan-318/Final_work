Тема: Разработка веб-приложения для системы автоматического управления (САУ) микроклиматом теплицы.
Цель: Создать веб-приложение, способствующее эффективному управлению микроклиматом в теплицах
Репозиторий из папки: Final_work.
Проект:
	django-admin startproject application_acs
	
	python manage.py startapp climate_management - Первым компонентом станет модель управления климатом, которая будет включать параметры температуры, влажности и освещения. Эта модель позволит хранить и обрабатывать данные о текущем состоянии микроклимата, здесь будет модель данных, представляемая в файле models.py, где будут описаны классы, отвечающие за такие сущности, как Температура, Влажность и Освещённость. Каждый класс будет включать соответствующие атрибуты и методы для управления данными.
	
	python manage.py startapp climate_track - Вторым компонентом станет система уведомлений - интерфейс пользователя, в виде веб-форм и отображений, которая будет оповещать пользователя о превышении заданных параметров микроклимата и предлагать рекомендации по оптимизации условий.
	
	python manage.py startapp users - Третьим компонентом станет приложение для разработчиков САУ.
	
	python manage.py startapp blog - Четвёртым компонентом станет приложение для хранения типовых решений по автоматизации САУ, предлагаемых разработчиками.
